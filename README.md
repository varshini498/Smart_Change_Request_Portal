Smart Change Request Portal is a full-stack web application that digitizes and streamlines the change request process within an organization. It allows employees to submit requests and managers to review, approve, or reject them through a structured dashboard.

Features

Secure login with role-based access (Employee / Manager)

Submit and track change requests

Approve or reject requests

Real-time status updates

Dashboard analytics

Export request data to CSV

Tech Stack

Frontend: React.js

Backend: Node.js, Express.js

Database: SQLite

Authentication: JWT
